## ğŸ§¾ SDS vs MSDS â€” What's the Difference?

| MSDS (Material Safety Data Sheet) | SDS (Safety Data Sheet)                  |
| --------------------------------- | ---------------------------------------- |
| Outdated format (pre-GHS)         | Modern, standardized format              |
| Inconsistent layout & language    | 16 fixed sections under GHS              |
| Less regulatory oversight         | Required for global hazard communication |

The SDS format, now standard under the **Globally Harmonized System (GHS)**, includes the following 16 sections:

1. Identification
2. Hazard(s) Identification
3. Composition / Ingredients
4. First-Aid Measures
5. Fire-Fighting Measures
6. Accidental Release Measures
7. Handling and Storage
8. Exposure Controls / PPE
9. Physical and Chemical Properties
10. Stability and Reactivity
11. Toxicological Info
12. Ecological Info
13. Disposal Considerations
14. Transport Information
15. Regulatory Info
16. Other Information
